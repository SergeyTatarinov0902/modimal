# Gulp сборка для верстки сайтов от Modimal

- HTML,
- SCSS,
- Gulp,
- JS

Документация и примеры использования внутри.

Установить зависимости:
```
npm i
```

Зпустить в режиме разработки:
```
gulp
```

Собрать версию для публикации:
```
gulp docs
```
